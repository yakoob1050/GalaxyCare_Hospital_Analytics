# Galaxy Care Hospital Analytics Dashboard

A Power BI dashboard providing insights on:

- Admit, Discharge, Follow-up Dates
- Billing Amount & Insurance Amount Comparison
- Bed Occupancy (Private, General, ICU)
- Doctor Feedback Volume
- Diagnosis Trends (Flu, Infection, Typhoid, Fracture)
- Patient-Level Financial Summary

## Tools Used
- Power BI
- Data Modeling
- Hospital Dataset
